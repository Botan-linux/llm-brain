# Ürün Bağlamı

## Ne Yapıyor
Kullanıcı mesajı alır → 16 modüllü pipeline'dan geçirir → düşünceli yanıt üretir. Basit bir chatbot değil; duygu işler, hafıza tutar, kişilik geliştirir, uyur ve rüya görür.

## Pipeline Sırası
1. Reflex → Acil/komut yanıt kontrolü
2. Language Processor → Dil analizi ve normalize
3. Thalamus → Giriş filtreleme ve yönlendirme
4. Limbic → Duygu işleme
5. Emotional Memory → Duygu hafıza eşleşme
6. Working Memory → Konuşma bağlamı yönetimi
7. Learning → Öğrenme ve desen tespiti
8. Subconscious → Bilinçaltı işleme
9. Self Awareness → Öz-farkındalık katmanı
10. Prefrontal → Karar verme ve planlama
11. Intelligence → Zeka katmanı (LLM entegrasyonu)
12. Ego → Benlik ve kişilik
13. Memory → Hafıza kaydı ve erişim

## Komutlar
- `uyu` / `sleep` → Uyku modu (konsolidasyon)
- `kapat` / `exit` → Kapatma
- `durum` / `status` → Durum raporu
- `hedefler` / `goals` → Hedef durumu
- `kimim` / `who am i` → Öz-farkındalık

## Hafıza Türleri
- **Working Memory**: Aktif konuşma bağlamı (JSON)
- **Emotional Memory**: Duygu-olay ilişkileri (JSON)
- **Long-term Memory**: Tarihli deneyim kayıtları (JSON + MD)
- **Personality**: Kişilik profili, zamanla gelişir (JSON)
- **Goals**: Hedef takibi (JSON)
- **Inner World**: İç dünya durumu (JSON)
- **Reflexes**: Otomatik yanıt kalıpları (JSON)
- **Prefrontal**: Karar verme durumu (JSON)
