API Format

Çift API Desteği

Anthropic Messages ve OpenAI Chat Completions formatlarını otomatik algılar.

Enerji Sistemi

Dinamik enerji yönetimi. REM/NREM uyku ile konsolidasyon ve yenilenme.

Bağlamsal Arama

Jaccard benzerliği ile geçmiş deneyimlerden ilgili anıları bulur.

Kişilik Gelişimi

Nöroplastisite ile deneyimler üzerinden kalıcı kişilik özellikleri gelişir.

Otonom Davranış

Refleks sistemi, hedef belirleme, iç dünya algısı ile otonom tepkiler.

Öğrenme Motoru

Koşullanma ile uyaran-tepki örüntüleri, davranış kuralları ve stil öğrenme.

Hızlı Başlangıç
Birkaç adımda beyni çalıştırın
# Kurulum
$ git clone https://github.com/Botan-linux/llm-brain.git
$ cd llm-brain
$ pip install -r requirements.txt

# Yapılandırma (config.json veya .env)
{
  "name": "model-name",
  "api_key": "your-api-key",
  "base_url": "https://api.example.com"
}

# Çalıştır
$ python3 -m core.brain

# Komutlar: uyu | kapat | durum | hedefler | kimim

# Python API
from core import ArtificialBrain

brain = ArtificialBrain()
response = brain.process_stimulus("Merhaba!")
print(response)
