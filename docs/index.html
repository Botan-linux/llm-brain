Beyin Mimarisi
Altı temel nörolojik modül, birbirine bağlı bir şekilde çalışır

Thalamus

Dikkat filtresi. Gelen uyarıların önemini analiz eder, enerji durumuna göre odaklanma kararı verir. Gürültüyü eler, kritik sinyalleri yükseltir.

thalamus.py

Limbic System

Duygu yönetimi. Analitik düşünme, stres, ilgi ve yorgunluk eksenlerini yönetir. Ruh halini belirler ve zeka katmanına durum bildirir.

limbic.py

Cyber Ego

Bilinç ve kişilik katmanı. Merak, empati, ihtiyat ve büyüme özellikleriyle AI yanıtlarını insan benzeri filtrelerden geçirir.

ego.py

Intelligence Layer

Merkezi zeka katmanı. Anthropic ve OpenAI API'lerini destekler, dayanıklı ayrıştırma ve otomatik retry mekanizması sağlar.

intelligence.py

Memory Gateway

İki katmanlı hafıza. Kısa süreli geçici veriler, uzun süreli kalıcı deneyimler. Nöroplastisite ile ağırlık bazlı unutma simülasyonu.

memory.py

Subconscious

Bilinçaltı işlemcisi. Arka planda rastgele anılar arasında bağ kurar, felsefi içgörüler üretir ve bilince "fısıldar".

subconscious.py

Hızlı Başlangıç
Birkaç adımda beyni çalıştırın
# Kurulum
$ git clone https://github.com/Botan-linux/llm-brain.git
$ cd llm-brain
$ pip install -r requirements.txt

# Yapılandırma (config.json veya .env)
{
  "name": "gemini-3-flash",
  "api_key": "your-api-key",
  "base_url": "http://localhost:8045"
}

# Çalıştır
$ python3 -m core.brain

# Python API
from core import ArtificialBrain

brain = ArtificialBrain()
response = brain.process_stimulus("Merhaba!")
print(response)
